User Interaction

Managing mutable variables, configuring local interaction event handlers, monitoring state flows, and triggering recomposition events cleanly.

Navigational Architectures

Designing structural application maps, passing parameters securely over routes, and implementing decoupled backstack patterns.

Core App Deployments

Project 1

StudyMate Interface Navigation

Problem Statement Students face significant academic fragmentation and struggle to manage runtime interface updates across separated views without losing user input or causing operational layout drift.

A multi-window academic planner engineered to isolate navigational flows while cleanly preserving operational input variables and application contexts globally using Compose-oriented architectural solutions.

Lessons Learned: Formulated a robust shared-state architectural scope across a multi-screen Jetpack Compose navigation host. Leveraged state hoisting strategies to preserve active text parameters and dynamically update layout hierarchies without forcing interface recomposition loops.

Project 2

StudyMate Telemetry & Databases

Problem Statement Local study setups lack active physical safety observation while cloud data metrics lack persistent remote backups, increasing student eye strain and leaving critical task telemetry data vulnerable to device losses.

An advanced utility dashboard processing continuous hardware illumination sensor streams, fetching real-time rest recommendations over endpoints, and routing student user logs into persistent Firebase databases.

Lessons Learned: Integrated on-device hardware Ambient Light Sensor hooks to issue real-time adaptive visual safety warnings. Constructed efficient Retrofit API clients to ingest continuous educational advice and mapped a secure Firebase Firestore pipeline to back up runtime telemetry objects live under my matric token document space.
